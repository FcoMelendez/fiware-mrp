Adopter Guide
=============

.. note::

   The Adopter Guide is being drafted and will be completed alongside
   Tutorial 12 (end-to-end demo and v1.0 release).

This guide is intended for organizations that want to deploy the FIWARE MRP
module against their own Orion-LD broker instance and integrate it with their
existing ERP or MES infrastructure.

Topics to be covered:

* Deploying to a production Kubernetes cluster
* Configuring multi-tenancy (Orion-LD tenants)
* Replacing the seed loader with a live ERP integration
* Extending the data model with custom terms
* Enabling NGSI-LD subscriptions for real-time events
