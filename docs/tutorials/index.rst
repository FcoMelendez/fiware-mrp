Tutorials
=========

Each tutorial introduces a limited set of concepts, delivers a runnable stack
increment, and ends with automated assertions you can verify yourself.
Every week builds on the previous one — but each tutorial directory is also
self-contained: reset, seed, execute, assert and stop.
